# Локальный запуск

## Быстрый старт

1. Запустите сервер:
   ```bash
   python manage.py runserver
   ```
   или двойной клик по `run.bat`

2. Откройте в браузере **http://127.0.0.1:8000/** (обязательно **http**, не https)

## Важно

- Файл `.env` уже создан с настройками для локальной разработки (DEBUG=1, секретный ключ).
- Сервер разработки поддерживает только **HTTP**. Если браузер открывает https:// — введите вручную **http://127.0.0.1:8000/**.
